Aseprite介绍

Aseprite诞生于2013年，是一款专业的像素动画制作软件。提供丰富多样的像素绘图工具，用户使用该软件就可以在电脑上制作出各种像素风格的2D动画，还支持将正常的图像转换成像素风格的图片，非常有趣好玩，是最流行的像素画制作软件。当然了万能的Photoshop也可以简单设置后就能进行创作像素画，相对于Ps阉割掉了对于像素画没用的功能，更加简洁明了，也更加适合没有软件基础的小白使用，上手门槛低。

如何下载呢？

  1. 官网：正式版20＄（可同时激活Steam版本），试用版免费（无法保存文件）。
  2. Steam：原价70¥，折扣48¥（经常打折），有钞能力的童鞋推荐以上两种方式。
  3. GitHub：Aseprite的源代码都开源发布在GitHub上，用户可以免费下载，然后手动编译成可运行的软件即可，编译方法在搜索引擎中很容易找到，GitHub上也有具体的编译方式，缺点就是比较麻烦，无法实时更新。

白嫖攻略（我不生产水,我只是大自然的搬运工）

编译Mac版本的 Aseprite，步骤如下：

  1. 确保电脑上已经装好了 XCode，并且启动过。
  2. 确保电脑上装好了 CMake，如果没装好，可以使用命令 brew install cmake 来安装。
  3. 确保电脑上装好了 Ninja，如果没装，使用命令 brew install ninja 来安装。
  4. 使用 git 克隆 aseprite 仓库。
    git clone --recursive https://github.com/aseprite/aseprite.git
  5. 下载 Skia，打开Github仓库，找到最新版本的 Skia-macOS-Release-X64.zip，下载后解压。
  6. 使用下面的命令，进入克隆好的 aseprite 目录，然后创建一个 build 目录，并进入 build 目录。
    cd aseprite
mkdir build
cd build
  7. 打开一个文本编辑器，对下面的命令进行一些修改。
    cmake \
  -DCMAKE_BUILD_TYPE=RelWithDebInfo \
  -DCMAKE_OSX_ARCHITECTURES=x86_64 \
  -DCMAKE_OSX_DEPLOYMENT_TARGET=10.9 \
  -DCMAKE_OSX_SYSROOT=/Applications/Xcode.app/Contents/Developer/Platforms/MacOSX.platform/Developer/SDKs/MacOSX.sdk \
  -DLAF_BACKEND=skia \
  -DSKIA_DIR=$HOME/deps/skia \
  -DSKIA_LIBRARY_DIR=$HOME/deps/skia/out/Release-x64 \
  -DSKIA_LIBRARY=$HOME/deps/skia/out/Release-x64/libskia.a \
  -G Ninja \
  ..
ninja aseprite
    • 在这种情况下， $HOME/deps/skia 改为解压后 Skia 的目录。
    • 确保 DMAKE_OSX_SYSROOT 指向 XCode 的 SDK 目录，但它在您的 Mac 中可能会有所不同，一般不会有问题。
  8. 复制上面修改好后的整个命令，然后在终端中执行即可开始编译。如果编译成功，则 bin 目录下的 aseprite 就是可执行文件。
  9. 如果您是苹果 M1 芯片，您可以按照与上述相同的步骤编译,不同的是下载 arm64 版本的 Skia，但是当我们调用时 cmake，有一些区别，如下。
    cmake \
  -DCMAKE_BUILD_TYPE=RelWithDebInfo \
  -DCMAKE_OSX_ARCHITECTURES=arm64 \
  -DCMAKE_OSX_DEPLOYMENT_TARGET=11.0 \
  -DCMAKE_OSX_SYSROOT=/Applications/Xcode.app/Contents/Developer/Platforms/MacOSX.platform/Developer/SDKs/MacOSX.sdk \
  -DLAF_BACKEND=skia \
  -DSKIA_DIR=$HOME/deps/skia \
  -DSKIA_LIBRARY_DIR=$HOME/deps/skia/out/Release-arm64 \
  -DSKIA_LIBRARY=$HOME/deps/skia/out/Release-arm64/libskia.a \
  -DPNG_ARM_NEON:STRING=on \
  -G Ninja \
  ..
ninja aseprite
